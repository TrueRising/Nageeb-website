# Nageeb Website

Static multi-page website for Nageeb.

## Main files
- `index.html` home page
- `about.html`
- `services.html`
- `why-us.html`
- `portfolio.html`
- `contact.html`
- `request.html`
- `thank-you.html`
- `story.html`
- `blog/` English blog
- `ar/` Arabic pages
- `assets/` CSS, JS, images

## Deploy on Netlify
- Build command: leave blank
- Publish directory: `/`

## Deploy on GitHub Pages
This project is plain static HTML/CSS/JS. Upload all files in this package to the root of your repository.
